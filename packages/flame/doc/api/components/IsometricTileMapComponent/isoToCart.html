Vector2 isoToCart (
  1. Vector2 p
)

Converts a coordinate from the isometric space to the cartesian space.

Implementation

Vector2 isoToCart(Vector2 p) {
  final x = (2 * p.y + p.x) / 2;
  final y = (2 * p.y - p.x) / 2;
  return Vector2(x, y);
}
