socketConnect method

Future<Socket> socketConnect (dynamic host, int port, { dynamic sourceAddress, Duration timeout })

Asynchronously returns a Socket connected to the given host and port.

When this override is installed, this functions overrides the behavior of Socet.connect(...).

Implementation 

Future<Socket> socketConnect(host, int port,
    {sourceAddress, Duration timeout}) {
  return Socket._connect(host, port,
      sourceAddress: sourceAddress, timeout: timeout);
}
