name property

String name

Get the name of the type, e.g. "IP_V4" or "IP_V6".

Implementation 

String get name {
  switch (_value) {
    case -1:
      return "ANY";
    case 0:
      return "IP_V4";
    case 1:
      return "IP_V6";
    default:
      throw new ArgumentError("Invalid InternetAddress");
  }
}
