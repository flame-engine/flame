Timeline class

Add to the timeline.

Timeline's methods add synchronous events to the timeline. When generating a timeline in Chrome's tracing format, using Timeline generates "Complete" events. Timeline's startSync and finishSync can be used explicitly, or implicitly by wrapping a closure in timeSync. For exmaple:

Timeline.startSync("Doing Something");
doSomething();
Timeline.finishSync();

Or:

Timeline.timeSync("Doing Something", () {
  doSomething();
});
