# Commands

The **commands** are special instructions surrounded with double angle-brackets: `<<stop>>`. There
are both built-in and user-defined commands.


```{toctree}
:hidden:

<<declare>>  <declare.md>
<<if>>       <if.md>
<<jump>>     <jump.md>
<<set>>      <set.md>
<<stop>>     <stop.md>
<<wait>>     <wait.md>
```
